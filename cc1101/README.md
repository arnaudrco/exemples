# Récepteur toute télécommande

<img width="664" height="557" alt="image" src="https://github.com/user-attachments/assets/ceceec6c-7755-4ac2-940f-7d7f6c9b6c6a" />

lorsque j'appuye sur un bouton d'une télécommande, le module me donne le code de la télécommande

<img width="275" height="83" alt="image" src="https://github.com/user-attachments/assets/44f42741-4ad4-4945-a5ad-be04308eda00" />

- VCC -> 3.3V
- GND -> GND
- MOSI -> GPIO13 (D7)
- MISO -> GPIO12 (D6)
- SCK -> GPIO14 (D5)
- CSN -> GPIO15 (D8)
- GDO0 -> GPIO5 (D1)
